# 换行
**在每段末尾使用两个空格**

**示例**  
a  
b  

```
a  
b  
```
  
# 缩进
使用占位符`&emsp;`(两格) `&ensp;`(一格) 可叠加使用

**示例**  
&emsp;a  
\&emsp;a  

&ensp;a  
\&ensp; a  

&emsp;&ensp; a  
\&emsp;\&ensp; a  

&emsp;&emsp;a  
\&emsp;\&emsp;a   

&ensp;&ensp;a  
\&ensp;\&ensp;a
